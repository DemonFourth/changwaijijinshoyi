### 详细计算过程

#### 1. 持有份额
```
持有份额 = 累计买入份额 - 累计卖出份额
        = 13,148.70份
```

#### 2. 持仓成本
```
持仓成本 = FIFO算法计算的当前持仓成本
        = ¥73,348.12
说明: 使用先进先出算法，扣除已卖出部分的成本
```

#### 3. 每份成本
```
每份成本 = 持仓成本 / 持有份额
        = ¥73,348.12 / 13,148.70
        = ¥5.5784
```

#### 4. 当前市值
```
当前市值 = 持有份额 × 当前净值
        = 13,148.70 × 8.0370
        = ¥105,676.10
```

#### 5. 浮动盈亏
```
浮动盈亏 = 当前市值 - 持仓成本
        = ¥105,676.10 - ¥73,348.12
        = ¥32,327.98
```

#### 6. 持仓收益率 ⚠️
```
持仓收益率 = 浮动盈亏 / 持仓成本 × 100%
          = ¥32,327.98 / ¥73,348.12 × 100%
          = 44.07%
```
**重要说明**:
- 这个收益率**只计算当前持仓的浮动收益**
- **不包含已实现的¥5,294.27收益**
- 这是正常的，不是错误

#### 7. 已实现收益
```
已实现收益 = 累计卖出获得的收益
          = ¥5,294.27
```

#### 8. 总收益
```
总收益 = 已实现收益 + 浮动收益
      = ¥5,294.27 + ¥32,327.98
      = ¥37,622.25
```

#### 9. 总收益率 ⭐
```
总收益率 = 总收益 / 总投入 × 100%
        = ¥37,622.25 / ¥100,000.00 × 100%
        = 37.62%
```
**重要说明**:
- 这个收益率**包含所有收益**（已实现 + 浮动）
- 这是**真实的投资回报率**

### 为什么持仓收益率 ≠ 总收益率？

这是**正常现象**，原因如下：

| 对比项 | 持仓收益率 | 总收益率 |
|--------|-----------|---------|
| 计算范围 | 只计算当前持仓 | 计算所有投资 |
| 包含已实现收益 | ❌ 不包含 | ✅ 包含 |
| 包含浮动收益 | ✅ 包含 | ✅ 包含 |
| 计算基数 | 持仓成本 | 总投入 |
| 用途 | 看当前持仓表现 | 看真实投资回报 |
