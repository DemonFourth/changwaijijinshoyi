# 清新现代风UI优化总结

## 优化目标
将基金计算器从之前的蓝紫渐变风格彻底转换为清新现代风（Notion/Airable风格）

## 配色方案

### 浅色主题
- **品牌色**：淡绿渐变 `#a8e6cf → #88d8b0`
- **买入/收益**：绿色 `#48bb78`
- **卖出/亏损**：珊瑚橙 `#ed8936`
- **分红**：天蓝 `#4299e1`
- **背景**：浅灰 `#f7fafc` / 白色 `#ffffff`
- **文字**：深灰 `#2d3748`

### 深色主题
- **品牌色**：深蓝绿渐变 `#1a365d → #2c5282`
- **买入/收益**：亮绿 `#68d391`
- **卖出/亏损**：浅红 `#fc8181`
- **分红**：亮蓝 `#63b3ed`
- **背景**：深蓝灰 `#1a202c` / `#2d3748`
- **文字**：浅灰 `#e2e8f0`

## 技术实现

### CSS变量体系
所有颜色都通过 `tokens.css` 中的CSS变量定义，支持主题切换：
- `:root` - 浅色主题
- `[data-theme="dark"]` - 深色主题

### 关键变量
```css
--color-brand-gradient: linear-gradient(135deg, #a8e6cf 0%, #88d8b0 100%);
--color-success: #48bb78;  /* 绿色 */
--color-danger: #ed8936;   /* 珊瑚橙 */
--color-info: #4299e1;     /* 天蓝 */
--color-text-on-brand: #2d3748; /* 品牌色上的文字 */
```
